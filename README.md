# ✍️ AI 汉字笔顺学习助手 | AI Hanzi Stroke Order Learner

2026年春节为孩子开发的轻量级汉字学习工具。一键模拟手写动画，搭配拼音与读音，让学习更有趣。
A lightweight Hanzi learning tool developed for kids during the 2026 Spring Festival. It simulates handwriting animations, displays Pinyin, and provides voice pronunciation to make learning fun.

## ✨ 功能特点 | Features
- **动态笔顺 (Stroke Animation)**: 模拟真实书写轨迹，一笔一划清晰可见。 (Simulates real writing paths with step-by-step animations.)
- **拼音标注 (Pinyin Support)**: 自动标注汉字拼音与声调。 (Automatically displays Pinyin with tone marks.)
- **语音朗读 (Voice Feedback)**: 浏览器原生语音合成，发音标准。 (Uses Web Speech API for clear character pronunciation.)
- **田字格视觉 (Standard Grid)**: 遵循传统教学格式，培养结构感。 (Standard "Tian Zi Ge" layout for structural practice.)

## 🚀 快速开始 | Quick Start
1. 在输入框输入任意汉字。 (Enter any Chinese character in the input box.)
2. 点击“开始”观看动画。 (Click "Start" to watch the stroke animation.)
3. 点击“听读音”跟读。 (Click "Audio" to listen and repeat.)
